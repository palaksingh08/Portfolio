## 📌 About This Project

This is my personal **developer portfolio website** created to showcase my skills, projects, achievements, and contact details in a modern and professional way.

It is designed with a clean UI, responsive layout, animations, and recruiter-friendly presentation.

## ✨ Features

* 🎨 Modern UI Design with gradients & glassmorphism
* 📱 Fully Responsive (Mobile / Tablet / Desktop)
* ⚡ Smooth animations & transitions
* 👩‍💻 About Me section
* 🛠 Skills showcase section
* 📂 Project cards with GitHub links
* 📄 Resume download button
* 📬 Contact section
* 🌙 Attractive developer theme
